CSS

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files
  • CSS selectors are used to "find" (or select) HTML elements based on their element name, id, class, attribute, and more.

JavaScript

JavaScript
What is it
JavaScript is the programming language of HTML and the Web.
What does it do?
Programs the behavior of web pages.
JavaScript Errors
Try statement lets you test block of code for errors. Catch lets you handle the error. Throw lets you create custom errors.
Finally statement
This statement will allow you to execute the code, after you try & catch, regardless of the result.
Debugging
Built-in debuggers can be turned on and off, forcing errors to be reported to the user.
